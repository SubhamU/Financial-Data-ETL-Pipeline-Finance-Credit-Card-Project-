NOTE: This project is referenced with different titles in resumes based on role focus (Data Analyst / Data Engineer). The repository represents the complete end-to-end work.

## Project Overview
Built an end-to-end financial data pipeline to transform
raw transaction data into an analytics-ready data model
for reporting and analysis.

## Tech Stack
SQL Server, Python, Power BI

## Data Engineering Work
- Extracted financial data from structured sources
- Transformed and loaded data into SQL Server
- Designed fact and dimension tables for analytics
- Applied data validation and quality checks

## Analytics & Reporting
- Created Power BI dashboards for financial KPIs
- Analyzed spending patterns and revenue trends
- Enabled consistent and scalable financial reporting
